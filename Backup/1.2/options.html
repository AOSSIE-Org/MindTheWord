Use the following user-defined translations:

For every pair of words "x":"y" in the comma-separated list above, the word "x" will always be translated to "y". You can use this in two ways:

Firstly, you can specify a list of words you have already learned and would always like to see translated
(for example, if you are learning Portuguese, and you have learned that "sky" is "céu", and you would like "sky" to always be translated to "céu", then add "sky":"céu" to the list above).
Secondly, you can prevent certain words from being translated by specifying that they should be translated to themselves (for example, if you don't want the article "the" to be translated, then add "the":"the" to the list above).
